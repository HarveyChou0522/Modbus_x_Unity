# Virtual-Physical Integrated Smart House

## Intro
This project used modbus TCP to connect virtual world and real world to sync the real-time data which can use for data analysis or actively control devices to maintain synchronization between physical and virtual environments.

## Feature
You can control the character in this game to interact the interiors in the scene. It would automatically pass the state of the device into the real world PLC,which would update the corresponding devices.
